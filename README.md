# Kviku Investor
1. git clone git@gitlab.com:lepsheycode/kviku-investor.git

2. cd kviku-investor

3. npm i && bower i

4. gulp



To create block run:

> blockAdd.sh block-name -T scss -T jade -T js -T css

To delete block run:

> blockDel.sh block-name



P.S. For sublime write ("atomic_save" : true) in Settings-User file.
